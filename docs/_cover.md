# Meadow Migration Manager <small>0.0.1</small>

> Database schema migration management for the Meadow ecosystem

- **Schema Libraries** -- manage and persist collections of MicroDDL schemas
- **Compilation** -- compile MicroDDL via Stricture to structured schema objects
- **Schema Diffing** -- detect table, column, index and foreign key changes between schemas
- **Migration SQL** -- generate database-specific DDL for MySQL, PostgreSQL, MSSQL and SQLite
- **Flow Visualization** -- interactive pict-section-flow diagrams with field-level FK connections
- **Three Interfaces** -- CLI, Terminal UI and Web UI for every workflow
